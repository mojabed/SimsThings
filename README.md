<h1 align="center">Python Decompiler</h1>
I decided to enhance the decompiler script going around with threading and a GUI, this should (depending on the system) make decompiling the files faster and a little more stable.
If you want to decompile Sims 4's Python scripts just follow these steps:

* Place the exe in a directory of your choosing. (i.e. D:/sims\_modding/) and run the exe
* Press browse and find your Sims 4 folder (i.e. F:/Games/The Sims 4) then press "Run decompiler" and grab a coffee
* Once done you'll have a folder called EA in your working directory with all the decompiled python files
